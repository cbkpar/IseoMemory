# 📷 MEMORY

> **추억을 수집하는 감성 사진 앨범 컬렉션 게임**

사진 한 장에는 그 순간의 추억이 담겨 있습니다.

**MEMORY**는 사진을 수집하고, 성장시키며, 나만의 앨범을 완성해 나가는 웹 기반 카드 컬렉션 게임입니다.

---

## 🚀 실행 방법

1. 저장소를 Clone 합니다.

```bash
git clone https://github.com/cbkpar/IseoMemory.git
```

또는 ZIP 파일을 다운로드합니다.

2. `index.html`을 실행합니다.

또는 GitHub Pages에서 바로 플레이할 수 있습니다.

👉 https://cbkpar.github.io/IseoMemory/

---

## ✨ 주요 기능

### 📖 사진 앨범

* 챕터별 사진 수집
* 사진 상세 정보 확인
* 즐겨찾기 기능
* 메모 작성 기능
* 획득 이력(History) 확인

### 🎴 카드 뽑기

* 카드 뒤집기(Flip) 연출
* 희귀도별 테두리 및 Glow 효과
* NEW 카드 전용 연출
* 등급별 파티클 효과
* 카드 상세 보기 연동

### ⭐ 카드 성장

* 카드 레벨 시스템
* 공명(Resonance) 시스템
* 카드 효과(스킬) 적용

### 💾 데이터 저장

* LocalStorage 자동 저장
* 즐겨찾기 저장
* 메모 저장
* 카드 수집 현황 저장

### 📱 모바일 지원

* 반응형 UI
* 모바일 환경 최적화
* 터치 인터페이스 지원
